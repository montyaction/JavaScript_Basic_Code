The = Operator

The = assignment operator assigns a value to a variable.

var x = 10

The += Operator

The += assignment operator assigns a value to a variable.

var x = 10;

x += 5;

The -= Operator

The -= assignment operator assigns a value to a variable.

var x = 10;

x -= 5;

The *= Operator

The *= assignment operator assigns a value to a variable.

var x = 10;

x *= 5;

The /= Operator

The /= assignment operator assigns a value to a variable.

var x = 10;

x /= 5;

The %= Operator

The %= assignment operator assigns a value to a variable.

var x = 10;

x %= 5;
